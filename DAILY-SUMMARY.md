# Daily Summary Fallback

Local file for daily notifications and EOD summaries when ClickUp is not configured (Phase 1 testing).

This file auto-appends trading summaries and EOD snapshots. Check it daily for a quick recap of bot activity.

---

## Format

Each entry follows:
```
## YYYY-MM-DD — EOD

[summary from /daily-summary command]

---
```

Keep it ≤ 15 lines per day. No long narratives — just facts and action items.
